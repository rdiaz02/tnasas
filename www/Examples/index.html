Data sets

You can download nine (well, 10) data sets here. This is a 16 MB download, and is identical to the one available from our tech. report supplementary material page.

These data are ready to be downloaded to your machine and from there be sent to GeneSrF or Tnasas. For further details on the data sets, where they come from ,etc, see the original references (included in the tech. report and it supplementary material available href="http://ligarto.org/rdiaz/Papers/rfVS/randomForestVarSel.html">here).

(What are you supposed to do with a tar.gz archive? In Linux, if using a shell, create a directory for the data and decompress the file there: ej: 

 
mkdir ExampleData 
cd ExampleData mv
../data.sets.tar.gz .  
tar -zxvf data.sets.tar.gz
If you are using some GUI for file management, well, use whatever the GUI says to do. In Windows, WinZIP and WinRAR both understand tar.gz; the free and LGPL'ed 7-zip understads those, and many more, too.)
